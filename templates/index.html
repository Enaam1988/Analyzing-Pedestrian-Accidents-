Project Introduction

The introduction sets the stage for our project, "Enhancing Pedestrian Safety Through Data Analysis," with a compelling narrative on the critical importance of pedestrian safety in our increasingly urbanized world. It highlights the unfortunate reality of pedestrian accidents and underscores the need to understand the underlying factors and patterns behind them. Through a comprehensive analysis of pedestrian crash data, we aim to make our streets safer. The introduction outlines our mission, which is to utilize data visualization techniques to explore temporal, spatial, and severity aspects of pedestrian-involved incidents. This data-driven approach promises actionable insights that can guide targeted safety measures and interventions. Ultimately, our project seeks to empower communities, authorities, and planners to create safer urban environments for pedestrians.

Crisp Cartography: Exploring Crash Data with Leaflet and Flask API

We built a web-based map using the Leaflet library to visualize crash data obtained from a Flask backend API. We initialized the map with a specific view, added an OpenStreetMap base layer, and fetched crash data. We created markers for each crash location, displaying relevant information in pop-up windows. To enhance security, we dynamically loaded the Leaflet library from a CDN and initialized the map only after the library was fully loaded. This code demonstrates how to create an interactive map that visually represents location-based data using Leaflet and integrates it with a web application through API requests.

The Dashboard

We've created a dynamic dashboard that fetches data from URL and visualizes it using D3.js and chart.js library. The dashboard includes a dropdown menu populated with unique "crash_id" values, allowing users to select a specific crash for analysis. Upon selecting a crash ID, the dashboard updates with charts illustrating the distribution of severity, crash types, and pedestrian casualties for the chosen crash. Each chart provides valuable insights into the selected crash's characteristics, enhancing data analysis and decision-making capabilities.
