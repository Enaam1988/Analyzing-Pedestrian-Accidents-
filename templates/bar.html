Pedestrian Crash Distribution

The core element of the code is a dynamic bar chart built to visualize the distribution of severity levels within pedestrian crash data. This chart leverages the Chart.js library, known for its flexibility and ease of use in creating interactive and informative data representations. The bar chart's primary function is to dynamically display the severity distribution as users interact with a dropdown menu to select different crash types. Each time a new crash type is chosen, the chart updates in real-time, offering an insightful and visually appealing way to explore the impact of various crash scenarios on severity levels.

The bar chart implemented within the code represents a powerful tool for visualizing and analyzing pedestrian crash data. Its ability to dynamically respond to user selections from the crash type dropdown menu enhances the code's usability and value. With Chart.js as its foundation, the chart combines aesthetic appeal with functionality, making it an excellent choice for anyone seeking to convey complex data trends related to crash severity. This dynamic visualization not only aids in better understanding the data but also adds an interactive dimension to the analysis, ensuring its relevance and usefulness in the context of pedestrian safety research and decision-making.
