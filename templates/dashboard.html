Crash Data Dashboard

The dashboard is designed to present road crash data in an accessible and informative manner and offers users a dropdown menu to select a specific crash ID for analysis. Once a crash ID is chosen,the dashboard dynamically updates to display several essential charts . These charts include a severity distribution chart , revealing the distribution of crash severity levels , a crash type distribution chart showing the proportions of various crash types , and a pedestrian casualties distribution chart, which depicts the number of pedestrian casualties in the selected crash . This interactive dashboard harnesses external libraries like Chart.js for creating visually engaging and insightful charts , empowering users to explore and gain insights from the road crash data effortlessly.
